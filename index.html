Features

⚡

Custom Bootloader

Custom Multiboot kernel compatible with GRUB for reliable booting.

🎮

Modern Graphics

High-quality VESA VBE resolution (1024x768x32) with optimized double-buffering.

🖱️

Advanced Input

PS/2 keyboard and mouse drivers with interactive graphical cursor.

🪟

Interactive GUI

Windowing system with taskbar, icons, and focus management.

🧠

Memory Management

Kernel heap allocator (kmalloc) for dynamic resource management.

📁

VFS

Basic virtual file system for managing files in memory.
